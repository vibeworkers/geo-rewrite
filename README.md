# geo-rewrite (Desktop 버전)

원본 콘텐츠(URL·첨부·붙여넣기)를 SEO/GEO 최적화 원고로 **분석·재작성**하는 순수 프롬프트 스킬. Claude Desktop / claude.ai용.

## 사용법

대화에서 다음 중 하나로 입력:
- **URL**: `https://example.com/post 윤문해줘`
- **파일 첨부** 후 "SEO 최적화로 다듬어줘"
- **본문 붙여넣기** 후 "분석부터 해줄 수 있어?"

옵션(자연어): 모드("분석만"/"바로 재작성"), 목표("GEO·전환 중심"), 분량("20% 줄여").

## 두 가지 모드

- **분석** — 목표별 현황 진단 + 개선 우선순위 제안
- **재작성** — 윤문 체크리스트 적용 후 윤문본 + 변경 요약 생성

| 고유성 | 1차 경험·차별 각도 |

## 핵심 원칙

- **추정 금지** — 본문에 없는 사실은 지어내지 않음(`TODO: 확인 필요` 표시)
- **원본 보존** — 핵심 주장·사실·고유 경험 유지, 표현·구조·신호만 최적화
- **변경 이유 명시** — 변경 요약에 기록
- **업종 규제 점검** — 의료·금융·교육 등 금지 표현·필수 고지

## 설치 (Claude Desktop / claude.ai)

1. `geo-rewrite-desktop.zip` 준비
2. 설정 → Capabilities(기능) → Skills → 스킬 업로드
3. `geo-rewrite-desktop.zip` 선택 후 업로드

> Skills는 유료 플랜(Pro/Max/Team/Enterprise)에서 제공됩니다.

## 범위 참고

이 스킬은 콘텐츠 한 편에 내재한 규칙만 적용하는 **Tunnel Framework Minimal 버전**입니다. 페르소나·CEP·프로젝트 코퍼스 near-duplicate 등 프로젝트 데이터가 필요한 최적화는 Tunnel Framework `/cp` 파이프라인에서 처리합니다.

## 관련 스킬

- `/cp` — CP 파이프라인(콘텐츠 생성·검증·배포, 풀 버전)
- `/schema-gen` — 구조화 데이터(JSON-LD) 생성
- `/geo` — GEO 감사(최적화 효과 검증)

> **Desktop 전용 스킬**입니다. Claude Code 슬래시 명령어로는 등록되지 않습니다(저장소 루트에 SKILL.md 없음).
